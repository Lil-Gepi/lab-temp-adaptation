This repository contains analysis scripts for the manuscript on laboratory 
and temperature adaptation in *Drosophila simulans*. 

## Structure 

- `Genomic`: genomic analyses 
	|- Data
	|- Script
	|- Plot

- `Phenotypic`: phenotype analyses
        |- Data 
	|- Script
        |- Plot

- `Transcriptomic/Script/`: RNA-seq analyses 
        |- Data 
	|- Script
        |- Plot

Generated plots, 
intermediate R objects, and large sequencing files are not tracked in this 
repository. 

## Data availability
Large raw data files are deposited separately and will be linked here.
